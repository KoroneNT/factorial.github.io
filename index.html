Aproximación de Stirling para factoriales grandes

La aproximación de Stirling es una fórmula utilizada para aproximar el valor de factoriales grandes:

n! ≈ √(2πn) (n/e)^n

La aproximación se vuelve más precisa a medida que n aumenta.

Ejemplo: 20!

Usando la aproximación de Stirling, podemos calcular el valor aproximado de 20!:

20! ≈ √(2π20) (20/e)^20 ≈ 2.43 x 10^18

El verdadero valor de 20! es:

20! = 2.4329 x 10^18

El error es:

|2.43 x 10^18 - 2.4329 x 10^18| / 2.4329 x 10^18 ≈ 0.0012 or 0.12%

Como puede ver, la aproximación es muy precisa para factoriales grandes.
